使用深度强化学习进行目标定位  
陈亮波 16721506  
（
上海大学 计算机工程与科学学院，上海 200444）  
摘要：作者呈现了一个在场景中定位目标的主动检测模型。模型是基于具体类别的，允许代理关  
注候选区域当在识别目标物体正确位置的过程中。代理学着去改变有界区域的形状，使用一些简  
单的变换动作，通过自上而下的推理，为了决定目标物体的具体位置。这种推荐定位的代理使用  
深度强化学习去训练，在 VOC 2007 的数据集上做评估。作者表示通过推荐模型作为向导的代理  
有能力去定位单个物体，在分析了仅仅 11~25 个图片中的区域，并且可以获得最好的检测结果在  
所有未使用目标推荐窗口的目标定位系统中。  
关键词：目标定位；深度强化学习；候选区域

Active Object Localization with Deep Reinforcement Learning  
Chen Liang-bo 16721506  
(School of Computer Engineering and Science, Shanghai University, Shanghai 20044, China)  
Abstract: The author present an active detection model for localizing objects in scenes.  
The model is class-specific and allows an agent to focus attention on candidate regions  
for identifying the correct location of target object. This agent learns to deform a bounding  
box using simple transformation actions, with the goal of determining the most specific  
location of target objects following top-down reasoning. The proposed localization agent  
is trained using deep reinforcement learning, and evaluated on the Pascal VOC 2007  
dataset. The author show that agents guided by the proposed model are able to localize a  
single instance of an object after analyzing only between 11 and 25 regions in an image,  
and obtain the best detection results among systems that do not use object proposals for  
object localization.  
Key words: object localization; deep reinforcement learning; candidate regions  
近几年来，深度学习兴起，在很多领域都获得了巨大的成功。特别是在图像处理领  
域卷积神经网络以它强大的学习能力击败了所有其他的机器学习算法，例如曾经最有效  
的支持向量机。并且我们惊讶于卷积神经网络的学习能力，它不同于传统的机器学习模  
型，它不需要人工去提取图像的特征（实际上人往往并不能有效地提取图像的特征），  
这种端到端的模式使得卷积神经网络自己去学习图像的特征。  
虽然卷积神经网络在图像的特征提取上，具 有无可比拟的优势，但是它并不能获得  
图像中物体的具体位置信息。于是有相当多论文采用滑动窗口的技术在图中搜索物体的  
位置，这种方法往往比较暴力，并且最后得到只是大量的推荐窗口，本质上并没有获得  
目标的位置信息，当然也有论文在此基础上改进，通过训练强分类器，例如支持向量机，  
对最后获得的推荐窗口评分，选取分数最高的作为最有可能包含物体的窗口。于是也有  
1
[6]  
人结合卷积神经网络和支持向量机做目标定位，例如 R-CNN 获得了非常高的检测率。  
然而，这些方法仍然是基于推荐窗口，进行目标定位的，并且推荐窗口数量仍然很  
大，例如 R-CNN，采用了 2000 个推荐窗口。于是这篇文章的作者并不满足这种基于推  
荐窗口的方法，尝试使用强化学习的方法来训练代理，使得代理可以通过学习到的知识  
来定位物体的具体位置。  
Google 的 DeepMind 团队，是最先将深度学习和强化学习结合起来的，他们认为真  
正的人工智能应当等于深度学习加上强化学习，于是出现了深度强化学习这个词汇。并  
且 DeepMind 团队利用他们的研究成果和他们自己开发一套框架 Deep Q-Network，先后  
在游戏领域获得了巨大成功，其中有两篇发在 Nature 上的论文，一篇是让计算机学习

发的 Alpha Go 向人类展示了它的超能力，以 4:1 压倒性的优势战胜了人类顶级围棋选  
手李世石。  
DeepMind 的这些成果，向人类展示了深度强化学习的无穷的潜力。因此，这篇论  
文的作者，在受到深度强化学习的启发下，决定将这项技术应用于图像的目标定位领域，  
在文章中，可以看出作者取得了较好的结果。本文将作为这篇论文的读书笔记，去感受  
和学习深度强化学习在图像目标定位中超强的能力。  
1
绪论  
1.1 研究背景与意义  
计算机视觉是使用计算机模拟人类视觉的学科，主要包括对目标进行识别、跟踪和测量  
等。由于图像的规模庞大，再加上早些年计算机硬件和处理速度受到限制，图像处理成为了  
一个瓶颈。直到 20世纪 70年代后期，计算机的性能才提高到可以处理图像这样大规模的数  
据，计算机视觉才开始发展。与此同时，人类对各种动物的眼睛、神经元以及与视觉刺激相  
关的脑组织进行了广泛的研究。伴随着神经生物学的发展，人类开始尝试使用计算机来模拟  
大脑的部分神经组织，各种模型层出不穷，其中最具有跨时代意义的应当是神经网络的提出，  
之后又陆陆续续被很多研究人员扩充和完善，最终使得计算机在一定程度上拥有了智能。  
所有的研究都是为了更好地服务人类社会，当然计算机视觉也不例外。人类试图让计算  
机去识别图像、分析场景，就是希望在将来计算机可以代替人类的眼睛，帮助人类无完成很  
多繁重的事情。例如摄像头的监视系统，现在世界各地安装了大量的摄像头，这意味着每时  
每刻都会产生无数的图像，而如果使用人力去监视这些图像，那么这个工作量将不能想象。  
试想如果计算机可以自己读懂这些摄像视频，并会分析视频中发生的事件，那么人类只需要  
为其二十四小时供电就可以了，什么地方发生了交通事故，什么地方出现了犯罪事件等等，  
人类都可以第一时间得知。这是一个非常理想的假想，但是我们相信在未来，或许是几百年  
或许是几千年，总会有那么一天，计算机将帮助人类完成很多工作。  
虽然本文只是研究计算机视觉中很小的一个部分——目标检测，但是从整个计算机视觉  
发展史的来看，本文的研究将促进计算机视觉向前发展，也为人类实现“计算机拥有真正意  
义上的视觉”这个终极目标贡献一份绵薄之力。  
1.2 研究现状  
在过去十年里，目标检测与定位已被视觉社区研究，朝着识别物体种类的目标，并且鉴  
别它所占的空间范围，利用紧的有界的盒子去覆盖所有它的可视区域。这是一个有挑战的计  
划要求对复杂的图形区域的计算与分析，一个好的任务经常被一个好的主意所驱动。  
在卷积神经网络(CNN)的帮助下，提高目标检测的精确率取得巨大的进步，凭借巨大的  
[6]  
视觉数据库和对图像类别的深度学习。一个成功的模型是 R-CNN 检测器，由 Cirshick 等人  
2
提出，结合了目标推荐和 CNN 的特点，实现最高水平的结果在 Pascal 和 ImageNet 检测中。  
其他一些工作也被推荐使用大容量的 CNN 去直接预测有边界的区域在回归背景下，并且取  
得了较好的结果。  
1.3 主要创新与特色  
在这篇文章中，推荐一个针对具体类别的有效的检测模型去学习定位一个目标物体，该  
物体是被系统所已知的。所提出的模型采取自顶向下搜索的策略，开始分析整个场景然后慢  
慢缩小到正确的目标位置。这个实现是采用了一系列对区域的变换，刚开始覆盖一个很大的  
区域，最终减少至一个紧的有边界的区域。这个变换序列通过代理分析当前可视区域的内容，  
然后选择下一步最佳的动作。每个转换应该保证目标在可视区域内，当在裁剪掉尽可能多的  
背景时。正如图 1 的一些动态决定步骤所示，在一幅图像上定位一头牛。  
图 1. 推荐算法采取的一个序列来定位一头牛，  
算法决定如何逐步变换区域来定位牛  
所提出的方法基本上不同于绝大多数的定位策略。与滑窗方式相反，本文的方法没有遵  
循一条固定的路径去搜索目标。相反，不同场景中的不同目标会以不同的搜索路径而结束。  
不像目标推荐算法，候选区域被选择通过高水平的推理策略，而不是低水平的线索。  
同时，与有边界的盒子的回归算法相比，本文的方法定位目标不是通过单一的、结构化的预  
测方式。作者提出一种动态的注意——动作策略，要求对当前区域的内容集中注意力，以这  
样一种方式变换盒子以至于目标物体可以逐渐被聚焦。  
为了提高所提出的代理的注意力，作者使用一个奖励函数与当前盒子覆盖目标表现相符。  
包含奖励函数在强化学习的背景下去学习定位策略，基于 Deep Q-Network 算法。作为一个  
结果，训练后的代理可以在 11 步内定位一个简单的实例，这意味着算法能够正确定位物体  
通过处理图像的仅仅 11 处区域。作者开展了一个综合实验评估在挑战 Pascal VOC 数据集  
上，获得了而一个具有竞争力的结果，就精度和召回率而言。
