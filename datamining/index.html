回归分析  
陈亮波  
（
上海大学，上海 200444）  
摘
要：回归分析是机器学习里非常简单而又实用的方法，本文将致力于讨论四种常用的回归模型，并利用这些模型解决  
一些实际的问题。机器学习里有很多重要的学习模型，其中两个重要的模型是回归和分类。本文将更关注于分类问题，针  
对手写数字识别和人脸识别两类问题，从概率学角度建模，利用逻辑斯蒂回归和 softmax 回归对其进行实验，并利用一些  
参数可视化的手段，对回归模型进行深入分析。实验结果显示，在手写数字集和人脸数据集，准确率均达到了 90%以上。  
关键词：分类、线性回归、多项式回归、逻辑斯蒂回归、softmax 回归  
Regression Analysis  
Chen Liangbo  
（
Shang University，Shanghai 200444）  
【
Abstract】Regression analysis is a very simple and practical method in machine learning. This paper will focus on the four  
commonly used regression models, and use these models to solve some practical problems. There are many important learning  
models in machine learning, two of which are regression and classification. This paper will pay more attention to the classification  
problem. For the problem of handwritten digital recognition and face recognition, the model is based on probability. Logistic  
regression and softmax regression are used to test it, and some parameter visualization methods are used to analysis regression in  
depth. Experimental results show that in the handwritten digit set and face data set, the accuracy rate has reached more than 90%.  
【
Key words】classification, linear regression, polynomial regression, logistic regression, softmax regression
